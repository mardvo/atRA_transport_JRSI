List of the files with explanations

1) main.m runs the model and produces figure 6. 

2) figure6d.m produces figure 6d, using output of main.m (needs to be saved as .mat files)

3) var_k1_k2.m makes parameter span for figure S1

4) opt_k2 find optimal value of k_prod and k_CYPdeg to fit experimental data. We first find k_CYPdeg for feeding condition and then k_prod for non-feeding.

5) Folder Sensitivity analysis contains codes for running sensitivity analysis and producing figure 7.
